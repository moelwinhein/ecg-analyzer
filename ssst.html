ECG Analyzer

MINDIST Analysis

In this analysis, MINDIST values are observed for different signal types (e.g. NA, NV, etc) of the same ECG signal.

Analysis Parameters

Median Filter: LEFT = RIGHT = 16
SG Filter: LEFT = RIGHT = ORDER = 8
Encoding: ALPHABET_SIZE = 8, PAA_SIZE = 32

Analysis

  • Signal 0 from MLII lead of SEL213 (QTDB) is filtered with the above mentioned filtering and encoding parameters.
  • For each of the RR-signals, its MINDIST with the other RR-signals in the ECG data signal is calculated and the results are plotted.
  • There are 17 RR-signal types in this ECG data piece as shown in the chart below.
  • Signals of the same type are in light-red color whereas other comparisons are in blue.
